# FinLens 财务比率计算公式

> **口径说明**：所有比率基于年度合并财务报表数据计算。涉及"平均"值的指标，使用（期初 + 期末）/ 2，但本工具用户仅提供当年及历史年度数据，因此当仅有当年数据时取年末值替代；有连续年度数据时取两年末平均值。

---

## 一、盈利能力 (Profitability)

### 1. 毛利率 (Gross Margin)
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%

分子：营业收入 - 营业成本（即毛利）
分母：营业收入
```
- **科目来源**：利润表
- **单位**：%
- **口径**：营业收入指主营业务收入，营业成本指主营业务成本。不扣除税金及附加。
- **N/A 场景**：营业收入 ≤ 0 时，标注"N/A（营业收入非正）"。
- **判断参考**：越高越好。跨行业差异大，一般 > 30% 视为良好。

### 2. 净利率 (Net Profit Margin)
```
净利率 = 净利润 / 营业收入 × 100%

分子：净利润
分母：营业收入
```
- **科目来源**：利润表
- **单位**：%
- **口径**：净利润为归属于母公司股东的净利润（含少数股东损益时取合并净利润）。
- **N/A 场景**：营业收入 ≤ 0 时，标注"N/A（营业收入非正）"。
- **判断参考**：越高越好。一般 > 10% 视为优秀。

### 3. 收入增速 (Revenue Growth Rate)
```
收入增速 = (当年营业收入 - 上年营业收入) / |上年营业收入| × 100%

分子：当年营业收入 - 上年营业收入
分母：上年营业收入的绝对值
```
- **科目来源**：利润表
- **单位**：%
- **口径**：同比增速。分母取绝对值以处理上年营收为负的极端情况。
- **N/A 场景**：无上年数据时，标注"N/A（仅一年数据）"。
- **判断参考**：正值为增长。需结合行业增速和利润质量综合判断。

### 4. 净利润增速 (Net Income Growth Rate)
```
净利润增速 = (当年净利润 - 上年净利润) / |上年净利润| × 100%

分子：当年净利润 - 上年净利润
分母：上年净利润的绝对值
```
- **科目来源**：利润表
- **单位**：%
- **口径**：同比增速。分母取绝对值以处理上年净利润为负的极端情况。
- **N/A 场景**：无上年数据时，标注"N/A（仅一年数据）"。
- **判断参考**：正值为增长。若收入增速低而净利润增速高，需关注是否是费用压缩或非经常性损益贡献。

### 5. ROE — 净资产收益率 (Return on Equity)
```
ROE = 净利润 / 平均净资产 × 100%

分子：净利润
分母：平均净资产 = (期初净资产 + 期末净资产) / 2
```
- **科目来源**：利润表 + 资产负债表
- **单位**：%
- **口径**：净资产 = 总资产 - 总负债 = 所有者权益合计。若仅有当年数据，分母取年末净资产。
- **N/A 场景**：净资产 ≤ 0（资不抵债）时，标注"N/A（净资产非正）"。
- **判断参考**：一般 > 15% 视为优秀，< 5% 为偏低。

### 6. ROA — 总资产收益率 (Return on Assets)
```
ROA = 净利润 / 平均总资产 × 100%

分子：净利润
分母：平均总资产 = (期初总资产 + 期末总资产) / 2
```
- **科目来源**：利润表 + 资产负债表
- **单位**：%
- **口径**：同上，仅有当年数据时取年末值。
- **N/A 场景**：总资产 ≤ 0 时，标注"N/A（总资产非正）"。（正常情况下总资产 > 0，此处为防御性处理）
- **判断参考**：一般 > 5% 视为良好。

### 7. 杜邦分解 (DuPont Analysis)
```
ROE = 净利率 × 总资产周转率 × 权益乘数

其中：
  净利率 = 净利润 / 营业收入              （分子：净利润，分母：营业收入）
  总资产周转率 = 营业收入 / 平均总资产    （分子：营业收入，分母：平均总资产）
  权益乘数 = 平均总资产 / 平均净资产      （分子：平均总资产，分母：平均净资产）
```
- **科目来源**：利润表 + 资产负债表
- **单位**：净利率（%）、总资产周转率（次）、权益乘数（倍）
- **口径**：三个因子乘积等于 ROE，用于分析 ROE 变动的驱动因素。
- **N/A 场景**：净资产 ≤ 0 时整体标注"N/A（净资产非正，杜邦分解不适用）"。
- **判断参考**：
  - 高 ROE 由高净利率驱动 → 品牌/技术壁垒强（最健康）
  - 高 ROE 由高权益乘数驱动 → 高杠杆（需关注偿债风险）
  - 高 ROE 由高周转率驱动 → 薄利多销型（需关注运营效率持续性）

## 二、偿债能力 (Solvency)

### 8. 流动比率 (Current Ratio)
```
流动比率 = 流动资产 / 流动负债

分子：流动资产
分母：流动负债
```
- **科目来源**：资产负债表
- **单位**：倍
- **口径**：流动资产含货币资金、应收账款、存货等；流动负债含短期借款、应付账款等。
- **N/A 场景**：流动负债为 0 时标注"N/A（无流动负债）"。
- **判断参考**：一般 > 2 为安全，< 1 为危险（流动资产不足以覆盖流动负债）。

### 9. 速动比率 (Quick Ratio)
```
速动比率 = (流动资产 - 存货) / 流动负债

分子：流动资产 - 存货
分母：流动负债
```
- **科目来源**：资产负债表
- **单位**：倍
- **口径**：剔除存货（变现能力最差的流动资产），更保守的短期偿债能力。
- **N/A 场景**：流动负债为 0 时标注"N/A（无流动负债）"。
- **判断参考**：一般 > 1 为安全，< 0.5 需关注。

### 10. 资产负债率 (Debt-to-Asset Ratio)
```
资产负债率 = 总负债 / 总资产 × 100%

分子：总负债
分母：总资产
```
- **科目来源**：资产负债表
- **单位**：%
- **口径**：总负债含流动负债 + 非流动负债。该指标衡量整体财务杠杆。
- **N/A 场景**：总资产为 0 时标注"N/A"（防御性，正常不会出现）。
- **判断参考**：一般 40%-60% 为适中，> 70% 为高杠杆（需结合行业）。金融行业不适用此标准。

### 11. 利息保障倍数 (Interest Coverage Ratio)
```
利息保障倍数 = (净利润 + 所得税费用 + 利息费用) / 利息费用

分子：净利润 + 所得税费用 + 利息费用（即 EBIT）
分母：利息费用（取自利润表"财务费用"科目中的"利息支出"）
```
- **科目来源**：利润表
- **单位**：倍
- **口径**：分子即 EBIT（息税前利润）。利息费用取 Excel 模板中单列的"利息费用"科目，对应用户利润表中财务费用项下的利息支出金额。若企业利息收入大于利息支出导致财务费用为负，利息费用仍取利息支出（正值）。
- **N/A 场景**：利息费用为 0 或未填列时，标注"N/A（无有息负债或利息费用未填列）"。
- **判断参考**：一般 > 3 为安全，< 1 为危险（息税前利润不足以支付利息）。

## 三、营运能力 (Efficiency)

### 12. 应收账款周转天数 (Days Sales Outstanding, DSO)
```
应收账款周转天数 = 平均应收账款 / 营业收入 × 365

分子：平均应收账款 = (期初应收账款 + 期末应收账款) / 2
分母：营业收入
```
- **科目来源**：资产负债表 + 利润表
- **单位**：天
- **口径**：应收账款含应收账款、应收票据。仅有当年数据时取年末值。**分母必须使用营业收入。**
- **N/A 场景**：营业收入 ≤ 0 时，标注"N/A（营业收入非正）"。
- **判断参考**：越短越好。与同行业比较，若显著高于同行需关注回款能力。

### 13. 存货周转天数 (Days Inventory Outstanding, DIO)
```
存货周转天数 = 平均存货 / 营业成本 × 365

分子：平均存货 = (期初存货 + 期末存货) / 2
分母：营业成本
```
- **科目来源**：资产负债表 + 利润表
- **单位**：天
- **口径**：存货含原材料、在产品、库存商品。仅有当年数据时取年末值。**分母必须使用营业成本。**
- **N/A 场景**：营业成本 ≤ 0 时，标注"N/A（营业成本非正）"。
- **判断参考**：越短越好。若显著延长可能意味着产品滞销或囤货。

### 14. 应付账款周转天数 (Days Payable Outstanding, DPO)
```
应付账款周转天数 = 平均应付账款 / 营业成本 × 365

分子：平均应付账款 = (期初应付账款 + 期末应付账款) / 2
分母：营业成本
```
- **科目来源**：资产负债表 + 利润表
- **单位**：天
- **口径**：应付账款含应付账款、应付票据。仅有当年数据时取年末值。**分母必须使用营业成本。**
- **N/A 场景**：营业成本 ≤ 0 时，标注"N/A（营业成本非正）"。
- **判断参考**：越长说明占用供应商资金能力越强（有利流动性），但过长可能意味着支付困难。

### 15. 现金循环周期 (Cash Conversion Cycle, CCC)
```
现金循环周期 = 应收账款周转天数 + 存货周转天数 - 应付账款周转天数

分子（逻辑上）：DSO + DIO - DPO
分母：不适用（直接加减）
```
- **科目来源**：上述三个指标汇总
- **单位**：天
- **口径**：衡量从支付采购款到收回销售款所需的总天数。
- **N/A 场景**：任一组成指标为 N/A 时，标注"N/A（组成指标数据不足）"。
- **判断参考**：越短越好。负值表示公司占用上下游资金运营（极强的话语权，如零售商超）。

## 四、现金流质量 (Cash Flow Quality)

### 16. 经营现金流 / 净利润
```
现金流净利润比 = 经营活动现金流量净额 / 净利润

分子：经营活动现金流量净额
分母：净利润
```
- **科目来源**：现金流量表 + 利润表
- **单位**：倍
- **口径**：衡量净利润中有多少是"真金白银"。经营现金流取自现金流量表中的"经营活动产生的现金流量净额"。
- **N/A 场景**：净利润为 0 或负时该指标无意义，标注"N/A（净利润非正）"。
- **判断参考**：
  - > 1：利润质量高，有真金白银支撑
  - 0.7-1：基本正常
  - < 0.7 且持续走低：利润"含金量"不足，可能存在应收挂账、存货积压等
  - < 0：严重预警，经营不产生正向现金流

## N/A 场景速查

| 场景 | 影响的比率 |
|------|-----------|
| 净资产 ≤ 0 | ROE、杜邦分解 |
| 营业收入 ≤ 0 | 毛利率、净利率、应收周转天数 |
| 营业成本 ≤ 0 | 存货周转天数、应付周转天数 |
| 利息费用为 0 或未填 | 利息保障倍数 |
| 净利润 ≤ 0 | 经营现金流/净利润 |
| 无上年数据 | 收入增速、净利润增速 |
| 流动负债为 0 | 流动比率、速动比率 |
